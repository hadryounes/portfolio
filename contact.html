What i can help you with

  • Ideation : I can help you with the exploratory phase of your project, I can do user research, brainstorming, competitive analysis and seek creative and innovative solutions for your complex design problems.

  • User experience design and documentation : metrics analysis, concept maps, personas, storyboards, wireframing, prototyping, design specifications...

  • User interface design : Style guides, High fidelity comp layouts, Responsive Guidelines, Reference guide, Interaction design...

  • Frontend development : design implementations using HTML5, CSS3, Jquery, SASS/Compass, Bootstrap, Foundation, pattern libraries, responsive validation...

  • Conversion Optimization : A/B Testing, Data gathering, Cross-browsers Validation, User Targeting...

  • Logo & identity design, Branding, Product Design Review, Design Consulting...

What i can't help you with

  • Backend developemnt : my coding skills allow me to build and deliver standards-compliant front-ends for web, however you will need a backend developer to help you make your product functional.

  • Advanced Javascript development : my JS skills are limited to implementing and customizing jquery-based plugins, I can't help you with any advanced JS work.

  • SEO, SMO and copywriting : my front-end code is highly optimized for SEO, however you will need a specialist to help you with web marketing.
